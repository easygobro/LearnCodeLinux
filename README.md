- Node.js (version 14 or above)
- npm (version 6 or above)

### Installation

```bash
git clone https://github.com/your-username/linux-learning-project.git
cd linux-learning-project
npm install
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
